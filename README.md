
# Satellite Crop Health

This project analyzes satellite imagery and weather data to monitor crop health and provide actionable alerts for farmers and stakeholders.

## Features
- Satellite data processing for crop health assessment
- Weather data integration for predictive analytics
- AI model for health prediction and alert generation
- Web-based frontend for visualization and user interaction
- Flask backend API for data handling and model inference

## Project Structure
- `backend/`: Flask backend, AI model, data processing, alerts, and session management
- `frontend/`: Static files and web interface
- `docs/`: Documentation and index.html
- `flask_session/`: Session files

## Getting Started
1. Clone the repository:
	```bash
	git clone https://github.com/vamsivalluri-19/satelite-crop-health.git
	```
2. Navigate to the backend directory and install dependencies:
	```bash
	cd backend
	pip install -r requirements.txt
	```
3. Run the Flask app:
	```bash
	python app.py
	```

## Usage
- Access the web interface via your browser.
- View crop health analytics and receive alerts.
